# 🛒 Sneakers E-Commerce Website

An elegant, responsive eCommerce product landing page designed to showcase sneakers. Built entirely with **HTML**, **CSS**, and **JavaScript**. This is a front-end-only project that includes interactive sliders, dynamic product selection, and a functional payment form.

## 🚀 Features

- ⚡ Interactive image slider for multiple product categories
- 🎨 Dynamic product updates (title, price, image, and color switcher)
- 📱 Fully responsive design with mobile-optimized layout
- 🛍 "Buy Now" button that triggers a stylish payment modal
- 🧾 Clean product listing with size and color options
- 📦 User-friendly UI, inspired by modern e-commerce platforms

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
- Media Queries for responsive design
